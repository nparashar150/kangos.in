Kang OS

Changelog

Download

KangOS is a sophisticated AOSP - based AOKP - inspired project, that focuses on redefining the stock Android interpretation and provide a more tinted and elegant user experience. KangOS focuses on DJ mix mash up of UI instead of boring stock visuals along with maximum stability and bug-less user experience.
